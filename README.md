## Running Project

1. Setup environment.

2. Install packages

```bash
npm install
```

3. Build the application

```bash
npm run build
```

4. Run application

```bash
npm run start
```

---

to run in dev mode:

```bash
npm run dev
```

## Tests

Run `npm test` in the project root to execute the unit tests via [Jest](https://jestjs.io).
